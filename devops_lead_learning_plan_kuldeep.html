Phase 1 — Foundation (weeks 1–3)
Wk 1
Cloud infrastructure — AWS & GCP Topic 1
Master VPCs, subnets, IAM roles, EC2/GKE compute, S3/GCS storage, load balancers. Learn how regions, AZs, and networking tie into your CI/CD pipelines.
AWS Skill Builder (free) GCP Fundamentals on Coursera Stephane Maarek – AWS DVA
Wk 2
Docker & containerizing GitHub repos Topic 4
Dockerfile best practices, multi-stage builds, .dockerignore, tagging strategy, pushing to ECR/GCR. Practice dockerizing one of your existing test automation repos.
Docker Docs – Get Started TechWorld with Nana (YouTube) Play with Docker (free lab)
Wk 3
CI/CD pipeline fundamentals Topic 3
GitHub Actions deep-dive (you already use this), then Jenkins & Azure Pipelines. Build a full pipeline: lint → test → build Docker image → push to registry → deploy to staging. Leverage your Selenium/Karate test suites as the test gate.
GitHub Actions docs Abhishek Veeramalla (YouTube) Azure DevOps free course – MS Learn
Phase 2 — Deep dive (weeks 4–8)
Wk 4–5
Kubernetes orchestration — EKS & GKE Topic 2 + 7
Pods, deployments, services, ingress controllers, namespaces, RBAC, config maps. Week 5: EKS cluster setup on AWS, node groups, managed add-ons. GKE Autopilot. Configuration changes on ASP.NET workloads running on EKS — replicas, resource limits, rolling updates.
KodeKloud – Kubernetes for Beginners AWS EKS Workshop (free) Killer.sh K8s labs
Wk 6
Helm charts Topic 5
Chart structure (Chart.yaml, values.yaml, templates), conditionals, helpers, overrides per environment. Design a Helm chart from scratch for one of your containerized apps. Package and push to a Helm registry.
Helm official docs TechWorld with Nana – Helm full course ArtifactHub.io examples
Wk 7
Terraform (IaC) Topic 6
HCL syntax, providers, modules, state management, remote backends (S3). Provision a VPC + EKS cluster end-to-end using Terraform. This is the most employer-tested skill in Canadian DevOps interviews — prioritize hands-on labs over video.
Zeal Vora – Terraform on Udemy HashiCorp Learn (free) terraform-aws-modules (GitHub)
Wk 8
Secure vaults & secrets management Topic 8
HashiCorp Vault — dynamic secrets, KV engine, auth methods (K8s auth, AWS IAM). AWS Secrets Manager and Parameter Store. Inject secrets into your pipelines and pods without hardcoding. Critical skill for banking/fintech — your TD background is a direct asset here.
HashiCorp Vault tutorials (free) AWS Secrets Manager workshop Vault on K8s – official docs
Phase 3 — Lead-level skills (weeks 9–10)
Wk 9
Observability & SRE practices Lead skills
Prometheus + Grafana, distributed tracing (Jaeger/Zipkin), log aggregation (ELK/Loki). SLIs, SLOs, error budgets. This is what distinguishes a Lead from an individual contributor — you own reliability, not just deployment.
Prometheus + Grafana – TechWorld Google SRE Book (free online)
Wk 10
Platform design & team leadership Lead skills
GitOps with ArgoCD, platform engineering concepts, defining golden paths for dev teams. Pair this with your existing Agile leadership — you already manage 18 QEs, so frame DevOps as platform ownership for engineering teams.
ArgoCD docs + crash course Platform Engineering on Medium

Portfolio projects — build these to show, not just tell
Project 1
Dockerized QA automation pipeline on GitHub Actions
Containerize your existing Selenium/Karate test suite → push to ECR → GitHub Actions triggers tests on every PR → test report published as artifact. This directly bridges your QA background into DevOps.
Project 2
Terraform + EKS: full infrastructure as code
Provision a VPC, EKS cluster, and node group on AWS using Terraform modules. Store state in S3. Document with README. This is your "show me IaC" answer in every DevOps interview.
Project 3
Helm chart with Vault-injected secrets on EKS
Deploy a sample app on EKS via Helm, inject DB credentials from HashiCorp Vault using the Vault Agent sidecar. Proves you can do orchestration + secrets together — the holy grail for banking DevOps roles.
Project 4
Observability stack: Prometheus + Grafana on K8s
Deploy Prometheus and Grafana via Helm on your EKS cluster. Define 3 SLOs for your test app. Build a Grafana dashboard. Screenshot it for your portfolio — visual proof of SRE thinking.
